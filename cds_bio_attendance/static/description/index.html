ZK Techo Biometric Machine Integration.

Fetch Biometric Logs From ZK Machines And Convert It To Attendances

Creative Digital Solutions SRL.

Features
  • Fetch Attendance Log From ZK Machines.
  • Machine Password protection.
  • Timezone Configuration and consideration for each machine.
  • Support Multiple type of punch.
  • Support Multiple Type Of Logs to Attendance Conversion (Actual check In/Check Out , First In/Last Out).
  • Support Fetching Start Date to ignore unwanted logs.
  • Minimum/Maximum Allowed Time Per machine .
  • Machine Connection log for troubleshooting.
  • Option For auto create employees with undefined Biometric ID.
  • Scheduled Action for auto download logs from machines.
  • State For Attendance (Right,Fix-in,Fix-out).
